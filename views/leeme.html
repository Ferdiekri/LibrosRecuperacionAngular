Pasos a seguir para ejecutar la aplicación

  1. npm i --> Instalar la carpeta "node_modules" con todas las librerías que necesitamos.
  2. npm run start --> Arrancar el proyecto junto a la BBDD.
  3. Terminal --> Ejecutar Tarea... --> tsc: seguir --> Para que nos compile los cambios guardados sin tener que arrancar de nuevo el proyecto.
