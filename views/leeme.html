Pasos a seguir para ejecutar la aplicación

  1. npm i Instalar la carpeta "node_modules" con todas las librerías que necesitamos.
  2. npm run start Arrancar el proyecto junto a la BBDD.
  3. Terminal Ejecutar Tarea... tsc: seguir - tsconfig.json Para que nos compile los cambios guardados sin tener que arrancar de nuevo el proyecto.

Información de la aplicación

Esta aplicación gestiona la venta y alquiler de viviendas y se compone de las suguientes secciones:

  • Inicio:El apartado principal. En ella se puede ver las diferentes viviendas y se puede buscar entre ellas la deseada.
  • Formulario:En este apartado se podrá dar de alta una nueva vivienda.
  • Resumen:Es el apartado en el que se saca cierta información de la base de datos.
  • Léeme:Fichero informativo que explica la aplicación.
