Necesitamos crear un gestor para poder escribir un libro de forma compartida. 

De un autor necesitamos saber su nombre y contraseña para acceder.

Un libro se compone de varias páginas. Tendrá una página inicial con el texto “Erase una Vez…” y autor “anónimo”.

Una página se compone obligatoriamente de un texto y un autor, cada página solo puede ser escrita por un autor y un autor puede escribir más de una página. Cada vez que un autor escribe una página se añade al final del libro.

La Aplicación como punto de entrada principal debe mostrar la 1º página del libro para que lo pueda leer cualquier usuario sin estar registrado. Desde esta pantalla se podrá avanzar y retroceder las páginas para seguir leyendo el libro. También debe existir la opción de ir a cualquier página deseada.

Para poder escribir una página el usuario debe hacer login en la App e iniciar una sesión.

   En el alcance de esta app no hace falta una gestión de usuarios. Pero sí que se deben realizar las siguientes tareas para un usuario que haya realizado el login:

·         Consultar todas sus página creadas

·         Crear nuevas páginas al final del libro

·         Editar cualquier pagina

·         NO se pueden eliminar paginas ya creadas
